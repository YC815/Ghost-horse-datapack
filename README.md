# [Minecraft-Datapack] <b>靈馬資料包<b/> by YC815
靈馬資料包 By.犽YC815
## <b> 👉 簡介：<b/>
本作品靈感來源來自《艾爾登法環》中的馬匹。<br>
只要製作靈馬召喚器，您就可以隨時召喚無敵馬匹。<br>
遠離靈馬後，靈馬將自動消失。<br><br>

## <b> 👉 使用說明：<b/>
- 靈馬召喚器合成配方：<br>
`|A|B|C|`<br>
`| |D| |`<br>
`|E|F| |`<br><br>
  >A: 終界水晶<br>
  B: 鞍<br>
  C: 鑽石馬鎧<br>
  D: 皮革<br>
  E: 地獄之星<br>
  F: 胡蘿波釣竿<br>
- 鑽石馬鎧合成配方:<br>
`|A|B|A|`<br>
`|A|C|A|`<br>
`|D|E| |`<br><br>
  > A: 鑽石<br>
  B: 皮革<br>
  C: 皮革馬鎧<br>
  D: 木棒<br>
  E:絲線<br>
- 使用流程：<br>
  > 製作靈馬召喚器後，手持召喚器點擊右鍵後將會召喚靈馬，對靈馬點擊右鍵即可乘坐。遠離靈馬3格以上後，靈馬將自動消失
